
# 📚 Fortbildung Projektmanagement + Scrum Master
### ChapterNext · 30.03.2026 – 12.10.2026

Dieses Repository enthält alle interaktiven Lerntools, die ich während meiner Weiterbildung zur **Projektmanagerin und Scrum Masterin** mit KI-Unterstützung entwickelt habe.

---

## 🗂️ Inhalt

| Datei | Beschreibung |
|---|---|
| `uebungen-ordner.html` | Interaktiver Übungsordner mit KI-Übungsgenerator – Herzstück der Lernbegleitung |
| `vokabelliste.html` | Wachsendes Fachglossar mit 37+ Begriffen aus PM, Scrum, KI & Digital – mit Lernmodus |
| `freitags-quiz-generator.html` | KI-gestützter Quiz-Generator – erstellt jeden Freitag automatisch neue Fragen zum aktuellen Modul |
| `meeting-notizen.html` | Interaktives Meeting-Protokoll-Tool mit Action Items, Export und Druckfunktion |
| `wochenquiz-30-03-2026.html` | Erstes Wochenquiz (Woche 1) – fest datiert, mit Ergebnisprotokoll |

---

## 🎯 Weiterbildungsinhalte

Die Fortbildung umfasst 21 Wochen und deckt folgende Module ab:

- **KI & Digitale Kompetenzen** – Algorithmen, neuronale Netze, LLMs, Prompt Engineering, KI-Ethik
- **Klassisches Projektmanagement** – Projektphasen, Stakeholderanalyse, PSP, Risikomanagement
- **Agiles Projektmanagement** – Scrum Framework, Sprint-Zyklen, Kanban
- **Scrum Master Zertifizierung** – Vorbereitung PSM I & PSM II
- **Digitale Tools** – Excel, PowerPoint, GitHub, KI-Tools im Büroalltag

---

## 🛠️ Technologie

Alle Tools sind als **standalone HTML-Dateien** gebaut – kein Server, keine Installation nötig. Einfach im Browser öffnen.

Die KI-gestützten Features (Übungsgenerator, Freitags-Quiz) nutzen die **Claude API** von Anthropic und funktionieren direkt im Browser.
